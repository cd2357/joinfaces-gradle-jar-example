# joinfaces-gradle-jar-example
JoinFaces Gradle Jar Example

Usages:
  - Can be started via its `main`-method using an embedded Tomcat
    - Using `java -jar`
    - Using `./gradlew bootRun`
    - Using your IDE
  - Can be tested using `@SpringBootTest(webEnvironment = RANDOM_PORT)`
    - Using `./gradlew test`
    - Using your IDE
